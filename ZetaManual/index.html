Overview

The Zeta AI Dashboard is a real-time command center for monitoring AI agents and managing tasks.

Purpose:

  • Monitor AI agents in real-time
  • Identify high-priority tasks instantly
  • Manage AI-driven workflows efficiently

Key Features:

  • AgentCards with status and task history
  • Priority badges: High ‚ö° / Medium üõ† / Low üí§
  • Filters by agent, priority, and keyword
  • Highlight-only mode for focused tasks
  • Toast + sound alerts for high-priority tasks
  • Panic mode for flashing high-priority tasks
  • Task acknowledgment to mark completed actions

Getting Started

System Requirements:

  • Modern web browser (Chrome, Firefox, Edge, Safari)
  • Access to the dashboard URL or local deployment
  • Audio enabled for sound alerts

Initial Setup:

  1. Open the dashboard in your browser
  2. Ensure `/alert.mp3` sound file is accessible
  3. Confirm agent data is streaming (AgentCards populate tasks automatically)

Interface Layout

Mission Control

Displays active/idle agent summary, filters, and live task list.

Highlight-Only Toggle

Shows only tasks matching active filters to reduce clutter.

AgentCards

Displays Name, Role, Task history, Priority badges, and Status.

Panic Mode Button

Toggle to flash all high-priority tasks and play continuous alert sound.

Notifications & Alerts

Toast Notifications

Appear in top-right for high-priority tasks and auto-dismiss after 5 seconds.

Sound Alerts

Triggered on high-priority tasks. Volume configurable (default 50%).

Panic Mode

Flashes high-priority tasks and plays continuous alert sound until acknowledged.

Task Acknowledgment

Each high-priority task has an Acknowledge button that stops flashing and dims the task. Panic mode continues for remaining unacknowledged tasks.

Tips & Best Practices

  • Monitor Mission Control regularly
  • Use highlight-only mode to focus on critical tasks
  • Activate panic mode only for multiple high-priority alerts
  • Adjust sound settings as needed
üí° Pro Tip: Combine filters with highlight-only mode during peak task periods for optimal focus.

Troubleshooting

  • No tasks visible: Refresh page and ensure agents are online
  • Sound not playing: Check browser permissions and volume
  • Tasks not updating: Verify network connection
